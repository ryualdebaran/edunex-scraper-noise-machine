# edunex-scraper-noise-machine
Tubes PRD

## How to Run:

1. Pastikan google chrome sudah login edunex
2. Nyalakan start.bat
3. (Sebaiknya) Tunggu sampai server menyala (ditandai dengan "BROADCASTING DEALINE SERVER:XXX.XXX.X.X:XXXX)
4. Nyalakan alat fisik
5. Enjoy (^_^)

## Wiring:

1. LCD-ESP32:
-GND to GND
-VCC to 5V
-SDA to 21
-SCL to 22

2. Buzzer-ESP32
-(+) to 18
-(-) to GND

Power source: Anything that can give 5V
